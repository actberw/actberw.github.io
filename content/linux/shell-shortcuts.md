Title: Linux shell 快捷键
Tags: linux, shell, bash shortcut
Date: 2014-04-13


ctrl + u 删除光标前面所有的
ctrl + k 删除光标后面所有的

ctrl + d 删除光标下得字符向后
ctrl + h 删除光标下得字符向前

ctrl + f向前移动 alt + f 单词
ctrl + b向后移动 alt + b 单词
ctrl + a 行首
ctrl + e 行尾
ctrl + y 粘贴
ctrl + w 删除前面得单词
Ctrl + xx - Move between EOL and current cursor position

iTerm2下用 `esc` 代替 `Meta` 键.
