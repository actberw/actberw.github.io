三角形的面积

Sun 10 August 2014

已知三边(a, b, c)求三角形面积.

海伦公式: 假设三角形三边分别为(a, b, c), 则三角形的面积
$S = \sqrt{p(p-a)(p-b)(p-c)}$, 其中$p=\frac{a + b+ c}{2}$

float space(float a, float b, float c) {
    int p;
    p = (a + b + c) / 2;
    return sqrt(p * (p - a) * (p - b) * (p - c));
}

已知平面三点的坐标求三角形的面积

利用向量叉乘 $S = \frac{1}{2}|\overrightarrow{AB}||\overrightarrow{AC}| * sign\angle{A} = \frac{1}{2}|\overrightarrow{AB} * \overrightarrow{AC}|$

typedef struct point {
    float x;
    float y;
} point_t;

float space(point_t A, point_t B, point_t C) {
    point_t AB = {B.x - A.x, B.y - A.y}, AC = {C.x - A.x, C.y - A.y};
    return fabs(AB.x * AC.y - AB.y * AC.x) / 2;
}
